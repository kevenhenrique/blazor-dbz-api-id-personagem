# BlazorDragonBallApi

Projeto desenvolvido para a lista de exercícios de Sistemas Distribuídos e Mobile.

## Objetivo

Criar uma aplicação em Blazor que recebe o ID de um personagem e consome o endpoint da Dragon Ball API:

`https://dragonball-api.com/api/characters/{id}`

A aplicação exibe:

- Nome do personagem
- Planeta de origem
- Transformações

## Como executar

1. Abra a pasta do projeto no Visual Studio ou Visual Studio Code.
2. Restaure os pacotes:

```bash
dotnet restore
```

3. Execute o projeto:

```bash
dotnet run
```

4. Acesse o endereço mostrado no terminal.
